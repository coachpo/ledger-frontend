# Ledger Workspace Frontend

React + Vite frontend for the portfolio-tracking MVP defined in `../docs/`.

## Scripts

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
pnpm test:e2e
```

`pnpm test:e2e` uses mocked API responses, so it does not require a live backend.

Copy `.env.example` if you want to point the app at a non-default backend URL.
